## 🎫 서비스 소개

<div>

팅팅은 티켓팅 시뮬레이션 서비스입니다. <br>
사용자에게 시뮬레이션을 통한 티켓팅 연습 (일명 포도알)을 제공합니다. <br>
서버는 한번에 몰리는 트래픽을 처리하기 위한 아키텍처를 사용합니다.
